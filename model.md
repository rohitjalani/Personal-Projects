# ML Model Training Pipeline
## Introduction
This repository contains the code for training a machine learning model. The following sections describe the structure and purpose of each file and directory in the repository.
## File Structure
model
train
demo
utils
constant
env
ppo
data
##Descriptions
**model**: File that contains the definition of the model.
**train**: Contains the training loop.
**demo**: Sample to use the code.
**utils**: Contains important functions like calculating rewards and advantages.
**constant**: Hyperparameters and problem definitions. 
**env**: Simulates the observation, action, and rewards pair.
**ppo**: Calculates PPO loss and performs the optimization.
**data**: Simulates the real-world dataset.
